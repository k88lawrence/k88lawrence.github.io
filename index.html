Projects

  • Plantstagram


    Plantstagram

    Single-Page Web App

    ◆Live ◆GitHub

    Plantstagram is a photo-sharing app inspired by the web version of Instagram made with Ruby on Rails and React/Redux. Users can upload their own photos, follow other users, and like and comment on posts. Photos are stored via AWS S3, making the app lightweight and scalable. The app make generous use of FlexBox for a seamless, responsive experience.

  • kQuery

    kQuery

    JavaScript DOM Manipulation Library

    ◆Live ◆GitHub

    kQuery is a JavaScript library that provides essential functionality for traversing and manipulating HTML documents. kQuery allows users to select and modify HTML elements, handle basic events, and make AJAX requests.

  • Sensory Input


    Sensory Input

    Interactive HTML5 Canvas/JavaScript Animations

    ◆Live ◆GitHub

    Sensory Input is a series of interactive HTML5 Canvas and JavaScript animations inspired by the five senses. Each visual element is created dynamically by JavaScript that scales to the window size and allows each page to be viewed optimally on screens of all sizes. Each element is animated with tweens from GSAP by Green Sock, allowing for movement that is smooth and visually attractive. Click and mouseover events allow users to interact with every element on the page.
